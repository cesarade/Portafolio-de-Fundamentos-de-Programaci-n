### Suma

Para la en python se utiliza el signo `+`, a continuación se muestra en ejemplo:

```python
suma = 1 + 3
print(suma)
[output] 4
```

Tambein, se pueden sumar con variables:

```python
# Se crean dos variables
a = 10; b = 40
suma = a + b
print(suma)
[output] 50
```
